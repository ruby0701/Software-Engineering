●安裝說明
----------------------------------------------------------------
請點擊"InstallSchoolHelper"，即會跳出視窗詢問是否安裝。
請按"是"。
即會啟動安裝精靈，選擇欲安裝的資料夾，
(預設資料夾為C槽的Program File(x86)的新建資料夾"NTPU CSIE")
接下來請連擊"下一步"，完成安裝。

若須解除安裝，則請點擊"Setup.exe"，選擇"解除安裝"。

----------------------------------------------------------------
●操作介紹
----------------------------------------------------------------


【第一頁】功課表輸入系統 :
請選擇您欲輸入的課程的日期，以及開始節數跟結束節數。
!!!請注意，課堂結束的時間不得早於課堂開始的時間!!!
接著，請您輸入課程名稱，並按下確認鍵，即成功新增課程!
若需要刪除課程，
則請輸入您要刪除的課程名稱，並選擇課程日期，以及開始節數跟結束節數
按下刪除後即刪除課程。

【第二頁】新增提醒事項 :
請選擇您的課程名稱，並輸入事項類型，
並選擇結束時間。
按下"提醒"之後即完成新增。
!!!請注意，當日無法新增當日的提醒事項!!!

【第三頁】缺席登記系統 :
請選擇您缺席的課程名稱，以及日期。
按下"缺席"鍵後即會出現紀錄。
並會幫您計算缺席的總次數。
下方並有缺席紀錄表可供您檢視自己的出缺席狀況。
